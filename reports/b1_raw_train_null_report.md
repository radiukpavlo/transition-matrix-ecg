## Reading Notes

- The five composite signature-score columns use the locked F24 form, but they are fully null in the current `B1_raw_train` artifact because the composite-score fitting stage has not yet populated them.
- Sparse nulls in interval-variability features are expected when a record lacks enough accepted beats or enough valid neighboring intervals to compute a stable statistic under the Appendix A rules.
- `qrs_axis_deg` is reported for clinician readability, while the fit-space path uses `qrs_axis_sin` and `qrs_axis_cos` to avoid circular-angle discontinuities.
- `qtc_formula_code` is metadata, not a fitted target. In the current run it is fully populated and records `QTcF`.
